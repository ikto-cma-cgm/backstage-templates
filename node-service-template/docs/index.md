# Node.js Microservice Template

Ce template fournit une base Node.js avec Express, TypeScript et l'intégration Backstage. Il génère un microservice prêt à être déployé avec SonarQube et les bonnes pratiques de monitoring.

## Comment utiliser

- Via la section **Create** dans Backstage
- En lançant le template **Node.js Microservice**

## Création d'un service

### Étape 1 : Métadonnées du service
- **Service name** : nom unique en kebab-case.
- **Description** : brève description du service.
- **Owner** : équipe responsable.

### Étape 2 : Placement dans le Catalog
- **System** : System Backstage rattaché.
- **Domain** : domaine métier.

### Étape 3 : Configuration build
- **Node.js version** : 18, 20 ou 22.
- **Express version** : version du framework Express.

Reuillez vos paramètres et cliquez **Create** pour générer votre microservice.
